NETIVE의 가치를 실현하는 사람

우리는 빠르게 변화하는 디지털 환경 속에서 공감으로 시작하고 (empathy),
정직한 협업으로 가능성을 확장하며 (honesty),
진정성 있는 태도로 한계를 넘어 함께 성장하는 (thoughtful) 사람과 같이 합니다.

  • EMPATHY
  • 공감할 줄 아는 사람
  • 사람에 대한 공감으로 시작하여
    전문성으로 완성할 줄 압니다.
  • HONESTY
  • 정직하게 협업하는 사람
  • 보여주기 위한 결과보다,
    과정의 진정성과 팀워크를 중시합니다.
  • THOUGHTFUL
  • 진정성으로 성장하는 사람
  • 깊이 있는 시선과 방향 있는 태도로
    지속 가능한 가치를 만들어 갑니다.

일과 삶의 균형을 존중하고,

같은 방향을 꿈꾼다면,
Netive에서 그 시작을 함께하세요.

    • Project Ownership
    • 프로젝트 전반을 기획부터 실행, 운영까지 총괄하며 일정·자원·이슈를 체계적으로 관리합니다.
    • 목표 달성을 위한 전반적인 진행 흐름을 리딩하며 결과물의 완성도를 책임집니다.
      Cross-team Collaboration
    • 기획, 디자인, 개발 등 다양한 팀 사이의 커뮤니케이션을 중재하고 협업을 이끕니다.
    • 각 파트의 이해관계를 조율하며 프로젝트의 일관된 방향성을 유지합니다.
      Domain-based Planning
    • 사용자와 서비스에 대한 맥락을 이해하고, 관련 정책과 기술 흐름을 고려해 기획에 반영합니다.
    • 도메인 지식을 바탕으로 실현 가능하고 지속 가능한 프로젝트를 설계합니다.
      Problem Solving
    • 프로젝트 진행 중 발생하는 다양한 이슈를 빠르게 분석하고 실행 가능한 해결책을 도출합니다.
    • 데이터를 기반으로 문제의 본질을 파악하고, 효과적인 대응 전략을 수립합니다.

    • User Research
    • 사용자 행동, 니즈, 사용 맥락 등을 정성·정량 데이터로 분석해 인사이트를 도출합니다.
    • 분석 결과를 바탕으로 더 나은 경험을 위한 기획 방향을 설정합니다.
      UX Strategy
    • 서비스의 방향성과 구조를 설계하며, 사용자와 비즈니스 간의 연결 고리를 만들어냅니다.
    • 데이터를 바탕으로 논리적인 기획을 설계하고 문제를 구조적으로 해결합니다.
      UX Planning
    • 사용자의 니즈를 중심으로 전체 서비스의 흐름과 화면 구성을 기획합니다.
    • 직관적이고 편리한 경험을 설계하는 것이 주요 역할입니다.
      Planning Communication
    • 기획 의도를 명확하게 전달하고, 유관 부서와의 긴밀한 협업을 통해 실행력을 확보합니다.
    • 다양한 관점을 조율하며 효율적인 의사결정을 이끕니다.

    • Design Thinking
    • 사용자의 니즈와 문제를 공감하고 정의하며, 창의적인 아이디어를 통해 솔루션을 도출합니다.
    • 프로토타이핑과 반복적인 테스트를 통해 더 나은 사용자 경험을 설계합니다.
      UI/UX Design
    • 사용자 흐름에 맞춘 UI/UX를 설계하고, 브랜드와 기능을 연결하는 시각적 경험을 만듭니다.
    • 다양한 플랫폼 환경에 적합한 일관된 디자인을 구축합니다.
      Design Collaboration
    • 기획자와 개발자와 함께 논의하며 피드백을 바탕으로 디자인을 완성도 있게 다듬습니다.
    • 실현 가능한 디자인을 만들기 위한 조율력을 갖춥니다.
      Creative Solving
    • 제한된 조건 속에서도 창의적으로 문제를 풀고, 새로운 접근으로 사용자 경험을 개선합니다.
    • 반복적 실험을 통해 더 나은 디자인 솔루션을 제안합니다.

    지금은 채용이 마감되었어요.

    • Frontend Dev.
    • Vue.js, React 등을 사용해 사용자와 직접 맞닿는 화면을 구현합니다.
    • 다양한 기기 환경에서 빠르고 안정적인 UI를 제공합니다.
      Markup Coding
    • 시맨틱한 HTML과 SCSS를 활용해 웹 표준에 맞는 UI 마크업을 구성합니다.
    • 반응형 설계 및 접근성을 고려한 구조를 구현합니다.
      Version Control
    • Git, SVN 등 형상관리 툴을 사용해 안정적인 협업과 코드 운영을 지원합니다.
    • 브랜치 전략과 코드 리뷰 문화를 이해하고 실천합니다.
      Web Standards
    • 웹 접근성과 퍼포먼스를 고려한 UI 구현을 통해 사용자 신뢰를 확보합니다.
    • 일관된 품질 기준을 바탕으로 완성도 높은 화면을 구축합니다.

    지금은 채용이 마감되었어요.

    • Tech Learning
    • 최신 기술과 도구를 적극적으로 탐색하고, 서비스에 적합한 솔루션을 제안합니다.
    • 변화에 유연하게 대응하며 기술 역량을 지속적으로 확장합니다.
      Issue Response
    • 운영 중 발생하는 이슈나 기술적 문제를 분석하고 근본적인 해결책을 구현합니다.
    • 안정적인 서비스 유지를 위한 실무적 해결력을 갖춥니다.
      Collaborative Dev
    • 다양한 직군과의 협업을 통해 기술적으로 조율된 결과물을 완성합니다.
    • 원활한 커뮤니케이션으로 개발 프로세스의 효율을 높입니다.
      Dev Leadership
    • 개발 과제를 주도적으로 추진하고, 전체 구조와 품질을 고려해 프로젝트를 이끕니다.
    • 적극적인 자세로 기술적 방향성을 제시합니다.
